# ✈️ ClearSkies: Airline Passenger Satisfaction Prediction

![Python](https://img.shields.io/badge/Python-3.8%2B-blue)
![Streamlit](https://img.shields.io/badge/Streamlit-App-red)
![Scikit-Learn](https://img.shields.io/badge/Scikit--Learn-ML-orange)

Bu proje, havayolu yolcularının uçuş deneyimlerini analiz ederek memnuniyet durumlarını tahmin eden bir makine öğrenmesi sistemidir.

## 📊 Veri Seti
- **Kaynak:** [Kaggle Airline Passenger Satisfaction](https://www.kaggle.com/datasets/teejmahal20/airline-passenger-satisfaction/data)
- **Kapsam:** 130.000+ yolcu kaydı, 25 farklı özellik.
- **Hedef:** Yolcu "Memnun" mu yoksa "Memnun Değil" mi?

## 🛠️ Kullanılan Algoritmalar
- **Klasik:** KNN, Naive Bayes, Decision Tree, Logistic Regression
- **İleri Seviye:** Artificial Neural Networks (ANN)
- **Optimizasyon:** Genetic Algorithm (GA)

## 🚀 Çalıştırma Talimatları
1. Depoyu klonlayın: `git clone https://github.com/yigitemre22/yapayzekaders.git`
2. Bağımlılıkları yükleyin: `pip install -r requirements.txt`
3. Uygulamayı başlatın: `streamlit run proje.py`
